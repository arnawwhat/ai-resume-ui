# 🚀 AI Resume Analyzer (Frontend)

A simple, modern React-based UI that allows users to upload a resume and a job description, and get AI-powered insights on how well they match.

---

## ✨ Features

- 📄 Upload Resume (DOCX / TXT)
- 📋 Upload Job Description
- 🤖 AI-powered analysis
- ⚡ Fast and minimal UI
- 🌙 Dark theme with a clean, modern design

---

## 🧠 What it does

This app connects to a backend API that:
- Compares resume with job description
- Calculates a match score
- Identifies missing skills
- Suggests improvements

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript
- CSS (custom styling)
- Fetch API
